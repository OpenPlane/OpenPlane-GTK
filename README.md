# OpenPlane

# Dépendances
- python3
- PyGObject
- matplotlib

# Installation
On commence par installer les dépendances :  
```sudo apt-get install python3-gi python3-matplotlib```  

Puis, on clone OpenPlane :  
```git clone https://github.com/boopr/OpenPlane.git```  

Enfin, on le lance :  
```python3 OpenPlane/app.py```
